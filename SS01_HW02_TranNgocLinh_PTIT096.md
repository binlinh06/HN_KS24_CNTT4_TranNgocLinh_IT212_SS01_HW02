# BÀI 2: Tối ưu Prompt (Giải nghĩa logic code Java)

## Prompt sau khi tối ưu

```text
Hãy giải thích ngắn gọn logic của hàm Java sau. Mô tả rõ điều kiện rẽ nhánh, giá trị đầu vào nào sẽ trả về "VIP" hoặc "Standard", và ý nghĩa nghiệp vụ của kết quả trả về.
```

## Đoạn phản hồi nhận được từ AI

```text
Hàm checkLevel(int points) dùng để xác định hạng thành viên dựa trên số điểm tích lũy của khách hàng.

- Nếu points lớn hơn hoặc bằng 1000, hàm trả về "VIP".
- Nếu points nhỏ hơn 1000, hàm trả về "Standard".

Về mặt nghiệp vụ, khách hàng đạt từ 1000 điểm trở lên sẽ được xếp vào hạng VIP, còn những khách hàng có số điểm thấp hơn sẽ thuộc hạng Standard.
```
